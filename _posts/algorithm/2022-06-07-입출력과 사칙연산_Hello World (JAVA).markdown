# 문제

 **Hello World!** 를 출력하시오.

## 입력

``` java
import java.lang.*;

public class Main{
    public static void main(String[] args){
        System.out.println("Hello World!");
    }
}
```

## 메모

앞으로 작성하게 될 모든 문서에서 참조해야할 듯하다.
